# TouHouThings

来为你的minecraft添加一点比较东方的东西吧！

目前已有的物品有：

·蓬莱药（给予使用者10分钟的抗性提升X、伤害吸收X、生命恢复X）

·阴阳玉（耐久为300，自带无限效果，可发射封魔针，也可用于近战）

·伊奘诺物质（目前用于合成阴阳玉和伊奘诺合金）

·御币（作为近战武器，小易易，死刑~）

·百百世之镐

·国士无双之药

·国士无双MAX（会让你飘起来）

·伊奘诺合金（目前仅用于合成百百世之镐）

推荐使用JEI物品管理器查看配方

本模组使Mcreator制作与打包

如果您有任何平衡性建议，欢迎提交issue
